# 🎓 Student Management System

A console-based Student Management System built using Python. This project demonstrates the fundamentals of Python programming, including functions, lists, dictionaries, loops, conditionals, exception handling, and CRUD operations.

## ✨ Features

* ➕ Add Student
* 👀 View All Students
* 🔍 Search Student by Roll Number
* ✏️ Update Student Marks
* ❌ Delete Student
* 🏆 Display Topper
* 📊 Calculate Average Marks
* ⚠️ Input Validation

## 🛠️ Technologies Used

* Python 3

## 🚀 How to Run

1. Clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the Python file:

   ```bash
   python student_management.py
   ```
